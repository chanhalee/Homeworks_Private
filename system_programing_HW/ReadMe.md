테스트 케이스 만들고 입력해주는 sh파일
===============================
### 실행방법
파일 두개를 자신의 코드가 있는 디렉터리에 다운 받기
해당 디렉터리에서 다음 명령입력
> ./test.sh

이후 실행결과와 test.sh내에 주석으로 표시된 기댓값과 비교
